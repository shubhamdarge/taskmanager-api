# Task Manager API

This project is a **Task Manager REST API** built with **FastAPI** and **Supabase**.  
It allows users to:

- Sign up and log in with email and password  
- Create, view, update, and delete personal tasks  
- Keep data secure using **JWT authentication** and **Row Level Security (RLS)** in Supabase  
- Explore all endpoints easily through the built-in **Swagger UI documentation**
